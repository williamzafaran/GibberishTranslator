## ✨ What it does:

### • One Hotkey. Done.
Select your gibberish, press your hotkey, and it's fixed. No retyping. No layout-switching.

### • Smarter Than Simple Scripts (Knows What You Meant)
Unlike basic AHK scripts that just swap keys, it figures out which layout produced the mess and converts it accurately. Arabic, Russian, Hebrew, Farsi, Greek, Hungarian, etc you name it — handles them all.

### • Cleaner Than Older Tools (Clipboard Stays Yours)
Doesn't pollute your clipboard with copied garbage. Uses zero CPU when idle and doesn't ask for admin rights. Just sits there, ready, waiting for you to mess up again.

### • Better Than Browser Extensions (Works Everywhere)
Unlike single-purpose extensions, it works natively system-wide: WhatsApp Desktop, Word, Notepad, Browsers, and VS Code. If you can type in it, Gibberish Translator can fix it.
